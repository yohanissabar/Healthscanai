# HealthScanAI - Modular PWA

Aplikasi pemantauan kebugaran biometrik berbasis Progressive Web App (PWA).

🌐 **LIVE DEMO:** [https://healthscanai.pages.dev](https://healthscanai.pages.dev)

## Arsitektur Modular
Proyek ini dibangun menggunakan Vanilla ES6 Modules demi performa maksimal tanpa bloatware framework.
